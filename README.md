# 📦 Customer Order Dashboard
> 🌟 An Order Management System built with **Flask** & **MySQL**

---

## 🌸 About The Project
A clean and simple web-based dashboard to manage and track customer orders.
Built using **Python Flask** for the backend and **MySQL** for the database! 💻

### ✨ What it shows:
- 💰 **Highest Value Order** — Rahul - ₹1,00,000
- 🔥 **Most Active Customer** — Rahul (1 orders)
- 📊 Order Table with Order ID, Customer, Product, Quantity, Price & Total

---

## 🛠️ Tech Stack
| 💡 Layer | 🔧 Technology |
|----------|--------------|
| 🎨 Frontend | HTML, CSS |
| ⚙️ Backend | Python, Flask |
| 🗄️ Database | MySQL |
| 🧩 Templating | Jinja2 |

---

## ⚙️ How to Run
1. Clone the repo
2. Run `pip install flask mysql-connector-python`
3. Setup MySQL database
4. Run `python app.py`
5. Open `http://127.0.0.1:5000`

---

## ✅ Features
- 📋 View all orders in table format
- 💰 Auto-detects highest value order
- 🔥 Finds most active customer
- ⚡ Dynamic data from MySQL
